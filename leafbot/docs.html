Get Started

Learn how to get Leaf on your server!

Installation

              
                1. Invite the bot, with the invite button, or with this link
                2. Make sure that the bot has administrator permissions in your server, either it wont, sadly, work
                3. Do .bothelp for a list of commands

Commands

Here are the commands and how you need to use them.

Command Meaing Ussage
.bothelp Show a list of avalabile commands in discord .bothelp
.ban Bans an user .ban @User#1234
.kick Kicks an user .kick @User#1234
.unban Unbans an user .unban User#1234
.clear Clears 1 or more messages. The maximum amount is 5 .clear **number**
.ping Verify your ping to the bot .ping
.play Play some music! .play MUSICNAME/td>
.stop Stop the music is playing now. .stop
.pause Pause the music it is playing right now .ping
.resume Resume the music from where you paused .resume
.leave Make Leaf get out the channel the music is playing :( .leave
.8ball Ask Leaf something .8ball 'question'

Tips: 1.You need to know the users tag you want to unban, because it wont work to unban him with an @User#1234 because he isnt in the server anymore
2. The default number for clear is 5. So if you will do .clear , it will clear 5 messages, including you .clear message!

Configuraton

Learn how to make Leaf recognize the permissions

I can start with an "It is quite simple" . Basically for every administrator role that you have, you need to do some tweaks:

For the command .ban to work, you need to give to the wanted administrator roles the Ban Members permission.

For the command .kick to work, you need to give to the wanted administrator roles the Kick Members permission.

For the command .unban to work, you need to give to the wanted administrator roles the Administrator permission. Be sure you trust them that time!

For the command .clear to work, you need to give to the wanted administrator roles the manage_messages permission.
